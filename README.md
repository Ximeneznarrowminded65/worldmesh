# WorldMesh
Official implementation of *WorldMesh: Generating Navigable Multi-Room 3D Scenes via Mesh-Conditioned Image Diffusion*. Code coming soon.

WorldMesh generates arbitrarily large multi-room 3D scenes efficiently using mesh-guided image diffusion.

[[arXiv](https://arxiv.org/abs/2603.22972)] [[Project Page](https://mschneider456.github.io/world-mesh/)] [[Video](https://youtu.be/MKMEbPT38-s)]
