## 🚀 What worldmesh does

worldmesh helps you create navigable 3D scenes from mesh-based image generation. It is made for people who want to turn a room setup into a multi-room world you can move through on a Windows PC.

Use it to:
- Generate room-sized 3D scenes
- Build connected spaces from image-guided mesh input
- Explore a scene with simple keyboard and mouse controls
- Export results for later use in 3D tools

## 🧩 What you can create

worldmesh is built for multi-room scene generation. It works well for:
- Indoor room layouts
- Connected hallways
- Simple apartment-like scenes
- Walkthrough previews
- Research demos
- 3D scene tests

The app uses mesh-conditioned image diffusion to help shape the final world. In simple terms, it uses an input mesh as a guide so the generated scene stays organized and walkable.
